Step 1: Update Repository Name
------------------------------
Edit gatsby-config.js:

  pathPrefix: '/YOUR-REPOSITORY-NAME',

‚ö†Ô∏è IMPORTANT: This must match your GitHub repo name exactly!

Examples:
  - Repo: "my-blog" ‚Üí pathPrefix: '/my-blog'
  - Repo: "personal-website" ‚Üí pathPrefix: '/personal-website'


Step 2: Create GitHub Repository
--------------------------------
1. Go to: https://github.com/new
2. Repository name: blog-app (or your choice)
3. Description: "AI-powered blog with dark theme"
4. Visibility: ‚úÖ PUBLIC (required for free GitHub Pages)
5. ‚ùå DO NOT initialize with README
6. Click "Create repository"


Step 3: Configure Git & Push
----------------------------
Run these commands in your terminal:

  # Configure Git (first time only)
  git config user.name "Your Name"
  git config user.email "your.email@example.com"

  # Add and commit files
  git add .
  git commit -m "Initial commit - AI Blog"

  # Connect to GitHub (replace USERNAME and REPO)
  git branch -M main
  git remote add origin https://github.com/USERNAME/REPO.git
  git push -u origin main


Step 4: Deploy to GitHub Pages
------------------------------
Run the deploy command:

  npm run deploy

Wait for completion (30-60 seconds).
This creates and pushes to gh-pages branch.


Step 5: Enable GitHub Pages
---------------------------
1. Go to your repository on GitHub
2. Click "Settings" tab
3. Click "Pages" in left sidebar
4. Under "Source":
   - Branch: gh-pages
   - Folder: / (root)
5. Click "Save"

Wait 2-3 minutes for GitHub to build your site.
